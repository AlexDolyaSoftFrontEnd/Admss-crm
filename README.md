# Admss-CRM

Это система управления взаимоотношениями с клиентами (CRM), предназначенная для автоматизации и оптимизации бизнес-процессов, связанных с работой с клиентами.

## Основные возможности

-   Управление клиентской базой
-   Ведение истории взаимодействий
-   Управление задачами и событиями
-   Аналитика и отчеты

## Установка

1. Клонируйте репозиторий:
    ```
    git clone https://github.com/yourusername/Admss-crm.git
    ```
2. Перейдите в папку проекта:
    ```
    cd Admss-crm
    ```
3. Установите зависимости:
    ```
    npm install
    ```
    или
    ```
    yarn install
    ```

## Запуск

Для запуска проекта используйте команду:

```
npm start
```

или

```
yarn start
```
